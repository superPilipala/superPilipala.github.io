市场分析

目标:

  • 形成关注市场的意识:happy:

  • 了解市场分析的几种方法:balance_scale:

  • 提升思考高度:thinking:

  • 激发创造力:ice_cream:

市场及用户分析:

背景:某互联网O2O平台老板/公司准备在宠物这个垂直类目寻找布局和深耕的机会,你会思考哪些?

  • 需要知道宠物人群规模,宠物规模,宠物商品/服务消费规模和趋势;(市场容量和趋势分析)
  • 什么人在养宠物?养宠物的人群特征? (用户分析)
  • 宠物主人都在养什么?宠物的分类?宠物的原因和需求? (对象细分)
  • 宠物消费在货梯交易,商品类,服务类的分布?养宠人群的信息关注? (需求分析)
  • 宠物市场行业从业者基本特征?宠物行业企业总体情况? (上下游分析)
  • 品牌/总代,批发/经销/代理,零售商的市场现状 (渠道商分析)
  • BAT和一些B+轮创业公司在宠物赛道的布局?(竞争格局分析)
  • 宠物消费有哪些趋势? (市场机会)
  • 自身有哪些优势和劣势? (优劣势分析)
  • 机会在哪里?准备在哪个细分领域正对哪些用户,满足哪些具体的场景的具体需求? (用户分析)
  • 什么时候开始开展项目? (进入市场的时机分析)
  • 需要投入多少钱?什么时候可以回本? (市场成本构成及回报周期分析)
  • 做这个生意/项目会不会有什么风险呢?潜在风险有哪些?如何应对? (风险和对策)
  • 期望用户量/营收趋势 (未来预测和展望)

市场容积分析:规模,趋势,BAT布局

  1. 市场容积分析一般包括:市场规模、增长率、预期、数据来源;
  2. 常见的市场报告下载:艾瑞、易观、IT桔子、比达网、百度文库;
  3. 通过一些媒体报道,能够相对快速了解BAT等巨头在相关细分领域的布局;
  4. 观察和沟通,观察身边相关线下门店的新开、转让情况,和身边朋友、从业者沟通;

自上而下
  • 源推算:本行业市场规模追溯到催生本行业的源行业.比如:通过房地产数据推演建筑材料、家具、家电等行业
  • 强相关:两个行业的产品销售一对一或者一对多的关系,比如:买了球拍需要买相应的求
  • 需求推算:相应目标人群或者需求较为明确且数据容易获取,比如:淘宝卖家增值服务的市场容量=商户数$\times$转化率$\times$客单价
自下而上
  • 抽样分析:在全国抽取1000户的样本,发现共有100户计划未来一年购买冰箱,每户一个,每个冰箱平均预算为2000,全国共5亿户,则冰箱的市场规模=100/1000*1*200*5亿=1000亿
  • 典型反推:行业演进经历三个阶段:极度分散、块状同质、团状异质.某个品牌的销售额通过多个渠道即可获取,而市场份额全凭个人经验,一般龙头企业的市场份额在3%~30%之间,具体数值视行业发展阶段而定

常见分析方法:波特五力,SWOT,PEST,波士顿矩阵

  • SWORT:优势、劣势、机会、威胁

    | | 优势(S) | 劣势(w) |
    | ——- | ——————————- | ——————————— |
    | 机会(O) | SO:优势+机会(快速发展且高风险) | WO:劣势+机会(利用机会,补齐弱点) |
    | 威胁(T) | ST:优势+威胁(利用优势,降低威胁) | WT:劣势+威胁(收缩合并做减法,守成) |

  • 4P/4C(战术策略):产品、价格、渠道、促销和消费者、成本、便利、沟通

  • PEST(宏观环境):政治、经济、社会、技术

    • 政治环境:国际关系、政治干预、方针政策、政治局势、国体与政体
    • 经济人口环境:宏观经济发展、经济基础结构、国家经济形势、经济发展水平、城市化程度、储蓄与信贷、消费结构、收入水平、人口变化
    • 技术自然环境:自然地理因素、科学技术发展
    • 社会文化环境:风俗习惯、审美观念、宗教信仰、价值观念、语言文字、教育水平

  • 竞争力分析:

    • 波特五力(行业环境):用户、供应商、竞争者、新进入、替代品
      • 潜在竞争者:进入时间和成本、进入所需行业知识、量产能力、成本优势、技术保护、进入壁垒:品牌政策等
      • 买家议价能力:客户数量、订单大小、同行业竞争对手差异性、价格敏感性、改变成本
      • 供应商议价能力:供应商数量、服务商大小、服务的独特性、创新能力、改变成本
      • 替代品威胁:替代品的业绩、改变成本
      • 同行竞争强度

  • 波士顿矩阵(产品决策):明星产品、现金牛业务、问题产品、瘦狗产品

    波士顿矩阵(BCG Matrix),又称市场增长率-相对市场份额、四象限分析法、产品系列结构管理法等

    | | 市场份额高 | 市场份额低 |
    | ———— | —————- | ————————- |
    | 市场增长率高 | 明星型业务:star: | 问题型业务:grey_question: |
    | 市场增长率低 | 现金型业务:cow2: | 瘦狗型业务:dog: |

市场分析频率:时刻,月度,年度

  • 时刻:政策、市场、竞对、媒体、论坛、使用
  • 月度:政策、市场、竞对报告、新版本、投融资
  • 专题:新业务成立/项目立项、市场、用户、需求、功能、运营

竞争力分析:

竞争力分析的两个前提:

  • 市场机会:赛道选取、用户洞察
  • 竞争对手:竞品选取、深度分析

其他一些竞争力分析方法:

  • 市场占有率分析:市场占有率通常用企业的销售量与市场的总体容量的比例来表示,同时关注细分市场占有率
  • 财务分析:竞争对手财务状况的分析主要包括:盈利能力分析、成长性分析和负债情况分析、成本分析等等
  • 创新能力分析:推出新产品的速度、科研经费占销售收入的百分比、销售渠道创新、管理创新
  • 竞争对手的领导人分析:对竞争对手领导人的分析包括:姓名、年龄、性别、教育背景、 主要的经历、培训的经历、过去的业绩等等.通过这些方面的分析,全面的了解竞争对手领导人的个人素质,以及分析他的这种素质会给他所在的企业带来什么样的变化和机会; 竞争对手主要的领导人的变更情况,分析领导人的更换为企业的发展所带来的影响
